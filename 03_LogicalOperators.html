Logical Operators

Logical operators are used to determine the logic between variables or values.

  • AND (&&) - Returns true if both statements are true
  • OR (||) - Returns true if one of the statements is true
  • NOT (!) - Reverse the result, returns false if the result is true

Logical AND (&&)

The AND operator is a binary operator. It takes two operands and returns true if both operands are true.

      
        A     B     A && B
        -------------------
        true  true   true
        true  false  false
        false true   false
        false false  false

Logical OR (||)

The OR operator is a binary operator. It takes two operands and returns true if one of the operands is true.

      
        A     B     A || B
        -------------------
        true  true   true
        true  false  true
        false true   true
        false false  false

Now we will see some examples in console:

        let a = true;
        let b = false;
        console.log("a || b : " + (a || b)); // true
        console.log("a || a : " + (a || a)); // true
        console.log("b || b : " + (b || b)); // false
Open developer tools of your browser to see the results in console.

Logical NOT (!)

The NOT operator is a unary operator. It takes one operand and returns the opposite boolean value.

      
        A     !A
        ----------
        true  false
        false true

Now we will see some examples in console:

        let a = true;
        console.log("!a : " + (!a)); // false
        console.log("!!a : " + (!!a)); // true
Open developer tools of your browser to see the results in console.

Logical Operators with Non-Boolean Values

Logical operators can be used with non-boolean values. In such cases, they return one of the operands based on their truthiness.

  • The AND (&&) operator returns the first falsy operand or the last operand if all are truthy.
  • The OR (||) operator returns the first truthy operand or the last operand if all are falsy.
  • The NOT (!) operator converts the operand to a boolean and then negates it.

Now we will see some examples in console:

        console.log(1 && 0); // 0
        console.log(1 && 5); // 5
        console.log(null && 5); // null
        console.log(0 && "hello"); // 0
        console.log(1 || 0); // 1
        console.log(null || "hello"); // "hello"
        console.log(null || 0 || 5); // 5
        console.log(!1); // false
        console.log(!0); // true
        console.log(!!"hello"); // true
        console.log(!!""); // false
Open developer tools of your browser to see the results in console.

Short-Circuit Evaluation

Logical operators use short-circuit evaluation, meaning they stop evaluating as soon as the result is determined.

  • For AND (&&), if the first operand is falsy, it returns that operand without evaluating the second.
  • For OR (||), if the first operand is truthy, it returns that operand without evaluating the second.

Now we will see some examples in console:

        function getValue() {
            console.log("getValue called");
            return 5;
        }
        console.log(0 && getValue()); // 0, getValue is not called
        console.log(1 || getValue()); // 1, getValue is not called
        console.log(1 && getValue()); // 5, getValue is called
        console.log(0 || getValue()); // 5, getValue is called
Open developer tools of your browser to see the results in console.

Practical Example

Logical operators are often used in conditional statements to combine multiple conditions.

        let age = 25;
        if (age >= 18 && age <= 65) {
            console.log("Eligible for work");
        } else {
            console.log("Not eligible for work");
        }
        let day = "Saturday";
        if (day === "Saturday" || day === "Sunday") {
            console.log("It's the weekend!");
        } else {
            console.log("It's a weekday.");
        }
        let isMember = true;
        let hasCoupon = false;
        if (isMember || hasCoupon) {
            console.log("You get a discount!");
        } else {
            console.log("No discount available.");
        }
        let isAdmin = false;
        if (!isAdmin) {
            console.log("Access denied. Admins only.");
        } else {
            console.log("Welcome, Admin!");
        }
Open developer tools of your browser to see the results in console.

Summary

Logical operators are used to perform operations on one or more boolean values.

  • AND (&&) returns true if both operands are true.
  • OR (||) returns true if at least one operand is true.
  • NOT (!) negates the boolean value.

They can also be used with non-boolean values, returning one of the operands based on their truthiness.

Logical operators use short-circuit evaluation to optimize performance.

They are commonly used in conditional statements to combine multiple conditions.

Understanding logical operators is essential for effective programming in JavaScript.
