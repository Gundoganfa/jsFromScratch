JavaScript Objects / Nesneler

Nesneler, değişkenlere benzer ama yalnızca değer değil, özellikler (properties) ve yöntemler (methods) de tutar. Ad–değer çiftleriyle veri saklayan konteynerler gibidir.

JavaScript’te Nesne Oluşturma

1) Object Literal

Tek seferlik nesneler için en pratik ve okunabilir yöntemdir.

<script>
            const person = {
            name: "John",
            age: 30,
            greet: function() {
            console.log("Hello, my name is " + this.name);
            }
            };
            person.greet(); // Hello, my name is John
            </script>

2) Constructor Fonksiyonu

Aynı türden birden fazla nesne üretmek için uygundur; ancak her örnek için metot kopyalanır.

<script>
            function Person(name, age) {
            this.name = name;
            this.age = age;
            this.greet = function() {
            console.log("Hello, my name is " + this.name);
            };
            }

            const p1 = new Person("John", 30);
            const p2 = new Person("Alice", 25);
            p1.greet();
            p2.greet();
            </script>

3) Class (ES6)

Constructor fonksiyonunun modern ve daha okunabilir karşılığıdır. Metotlar prototipte paylaşılır (daha verimli).

<script>
            class Person {
            constructor(name, age) {
            this.name = name;
            this.age = age;
            }
            greet() {
            console.log("Hello, my name is " + this.name);
            }
            }

            const person = new Person("John", 30);
            person.greet();
            </script>

Özet

  • Object literal: Tek nesne için kısa ve anlaşılır.
  • Constructor function: Çoklu üretim için kullanılabilir ama modern projelerde daha az tercih edilir.
  • Class: Modern, okunaklı ve bellek açısından verimli; güncel projelerde yaygın.
JavaScript’te Nesne Özelliklerine Erişim

Bir nesnenin özelliklerine iki şekilde erişebilirsiniz: nokta notasyonu veya köşeli parantez notasyonu.


            // Nokta Notasyonu
            console.log(person.name);
            person.greet();

            // Köşeli Parantez Notasyonu
            console.log(person["age"]);
            person["greet"]();

Yukarıdaki örnekte, person nesnesinin name ve age özelliklerine eriştik ve greet yöntemini çağırdık.

Özellik Ekleme ve Değiştirme

Bir nesneye yeni özellikler ekleyebilir veya mevcut özelliklerin değerini değiştirebilirsiniz.


            // Yeni özellik ekleme
            person.job = "Developer";

            // Mevcut özellikleri değiştirme
            person.age = 31;

Yukarıdaki örnekte, person nesnesine yeni bir job özelliği ekledik ve age özelliğinin değerini 31 olarak değiştirdik.

Örnek Kod


            const person = {
                name: "John",
                age: 30,
                greet: function() {
                    console.log("Hello, my name is " + this.name);
                }
            };
            console.log(person.name); // John
            person.greet(); // Hello, my name is John   
            person.job = "Developer";
            person.age = 31;
            console.log(person.age); // 31      
            console.log(person.job); // Developer

Summary / Özet

  • JavaScript nesneleri, adlandırılmış değerler (özellikler) ve işlevler (yöntemler) içeren veri yapılarıdır.
  • Nesne özelliklerine nokta notasyonu veya köşeli parantez notasyonu ile erişilebilir.
  • Nesnelere yeni özellikler eklenebilir ve mevcut özelliklerin değerleri değiştirilebilir.
JavaScript Nesneleri ile İlgili Faydalı Metotlar
  • Object.keys(obj): Verilen nesnenin tüm anahtarlarını içeren bir dizi döner.
  • Object.values(obj): Verilen nesnenin tüm değerlerini içeren bir dizi döner.
  • Object.entries(obj): Verilen nesnenin tüm anahtar-değer çiftlerini içeren bir dizi döner.

            const person = { //already defined above
                name: "John",
                age: 30,
                job: "Developer"
            };
            
            console.log("--- Useful Methods for JavaScript Objects ---");
            console.log(Object.keys(person)); // ["name", "age", "job"]
            console.log(Object.values(person)); // ["John", 30, "Developer"]
            console.log(Object.entries(person)); // [["name", "John"], ["age", 30], ["job", "Developer"]]
Döngüler ile Nesne Özellikleri Arasında Gezinti

JavaScript'te bir nesnenin tüm özellikleri üzerinde döngü yapmak için for...in döngüsü kullanılır. Bu döngü, nesnenin tüm enumerable (sayılabilir) özelliklerini tek tek işler.

                
                console.log("--- Looping through Object Properties ---");
                for (let key in person) {
                    console.log(key + ": " + person[key]);
                }
