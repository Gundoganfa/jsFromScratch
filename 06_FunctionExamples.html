Functions Examples in JavaScript

Here are some examples of functions in JavaScript:

Write in ? or || form. 
        function checkAge(age) {
          if (age >= 18) {
            return true;
          } else {
            return confirm('Do you have permission from your parents?');
          }
        }
Now we will see how to write above function in a shorter way using the ? (tenary) operator: 
        function checkAge(age) {
            return age>=18?true:confirm('Do you have permission from your parents?');
        }
Now we will see how to write above function in a shorter way using the || operator: 
        function checkAge(age) {
            return age>=18 || confirm('Do you have permission from your parents?');
        }
I personally prefer the middle one using the ? operator.

Here is another example of a function that calculates the minimum of two numbers: 
        function min(a,b) {
            return (a<b)?a:b;
        }
        console.log(min(2,5)); // 2
        console.log(min(3,-1)); // -1
        console.log(min(1,1)); // 1
Here is an example of an recursive Arrow Function that calculates the power of a number. Attention, the function calls itself in its body with a smaller exponent until the exponent is 1. 
        let power = (base, exponent) => {
            if (exponent === 1) {
                return base;
            } else {
                return base * power(base, exponent - 1);
            }
        };
Now let's see a function expression example, check the console for results: 
        let sayHello = function() {
            console.log("Hello!");
        };
        sayHello();
Note that the function has not a name, it is an anonymous function assigned to the variable sayHello.
check the console, we'll log sayHello variable: 
        console.log(sayHello); // shows the function code
