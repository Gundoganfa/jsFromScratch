Objects in JavaScript

Objects are used to store collections of data and more complex entities.

Objects are variables too. But objects can contain many values.

Objects are written with curly braces {}.

Object properties are written as name:value pairs (name and value separated by a colon).

        const person = {
          firstName: "John",
          lastName: "Doe",
          age: 50,
          eyeColor: "blue"
        };
        console.log(person);
        console.log(person.firstName); // John  
        console.log(person["lastName"]); // Doe
        person.age = 51; // change age
        console.log(person.age); // 51
        person.eyeColor = "green"; // change eye color
        console.log(person.eyeColor); // green
        person.nationality = "American";
        console.log(person.nationality); // American
