Callback Functions in JavaScript

A callback function is a function passed into another function as an argument, which is then invoked inside the outer function to complete some kind of routine or action.

Callback functions can be synchronous or asynchronous.

Example of a Synchronous Callback Function

processUserInput fonksiyonuna greet fonksiyonu callback olarak geçilir. processUserInput fonksiyonu içinde, callback isminde bir fonksiyon çağırılmış olmasına rağmen, tanım gereği greet fonksiyonu çağırılır. 
        function greet(name) {
            return 'Hello ' + name;
        }

        function processUserInput(callback) {
            const name = prompt('Please enter your name:');
            console.log(callback(name));
        }

        processUserInput(greet);

Example of an Asynchronous Callback Function

        setTimeout(function() {
            console.log('This message is shown after 2 seconds');
        }, 2000);
