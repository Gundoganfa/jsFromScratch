Yeni JavaScript Özelliklerini Eski Tarayıcılarda Çalıştırmak

JavaScript sürekli gelişiyor. Ancak eski tarayıcılar bu yeni özellikleri her zaman desteklemiyor. Bu sorunu çözmek için iki temel araç kullanılır:

  • Transpiler: Yeni sözdizimini eskiye çevirir.
  • Polyfill: Eksik fonksiyonları tanımlar.

Transpiler Nedir?

Modern JavaScript kodunu eski sözdizimine çeviren yazılımdır. Örneğin:

Önce: height = height ?? 100;

Sonra: height = (height !== undefined && height !== null) ? height : 100;

En popüler transpiler: Babel. Genellikle webpack gibi araçlarla otomatik çalıştırılır.

Polyfill Nedir?

Yeni JavaScript fonksiyonlarını eski tarayıcılarda tanımlamak için kullanılır. Örneğin Math.trunc() fonksiyonu yoksa:


if (!Math.trunc) {
  Math.trunc = function(number) {
    return number < 0 ? Math.ceil(number) : Math.floor(number);
  };
}

Popüler polyfill kütüphanesi: core-js

Özet

Modern JavaScript özelliklerini kullanmaktan çekinmeyin. Sadece transpiler ve polyfill araçlarını doğru şekilde entegre ettiğinizden emin olun.
